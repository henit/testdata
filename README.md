# testdata

Test data for unit tests and ui development. Because Lorem is boring!

# Usage

```shell
    npm install --save-dev testdata
```

# Game of Thrones

## All objects

```javascript
    import got from 'testdata/got';

    console.info(got.author, got.book1, got.TyrionLannister);
```
